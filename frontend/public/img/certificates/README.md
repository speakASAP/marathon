# Runtime Certificate Backgrounds

These PNG files are the runtime certificate backgrounds used by `frontend/src/pages/ProfileAwards.tsx`:

- `gold_en.png`
- `silver_en.png`
- `bronze_en.png`

They are generated from the canonical user-reference template stored in `docs/assets/certificate-template`.

Do not replace them with a newly drawn certificate style unless the owner explicitly approves a redesign. For coordinate data, transparent overlays, trophy assets, and the editable source template, use `docs/assets/certificate-template/certificate-layers.json`.
